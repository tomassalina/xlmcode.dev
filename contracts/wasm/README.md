Audited, pre-compiled Soroban contracts live here (PLAN.md §2, §9). They are
**configured** by the user and **deployed** by the backend via the JS SDK — never
compiled in the browser or on Vercel.

referenced by `contracts/manifests/oz-fungible-token.json` → `wasmPath`.
